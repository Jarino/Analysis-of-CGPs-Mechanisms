Analysis of Cartesian Genetic Programming’s Evolutionary Mechanisms
===============================================================================

These pages provide code documentation for the Analysis of Cartesian Genetic Programming’s Evolutionary Mechanisms paper published
IEEE Transactions on Evolutionary Computation by Brian W. Goldman and William Punch.
For information about how to run this code please see the
:mod:`main` module.  For support questions feel free to email brianwgoldman@acm.org.
